higher\n- PostgreSQL database\n- Supabase account (for real-time features)\n\n### **Installation**\n\n```bash\n# Clone the repository\ngit clone https://github.com/iMoovSmoov/astrology-platform.git\ncd astrology-platform\n\n# Install dependencies\nnpm install\n\n# Set up environment variables\ncp .env.example .env.local\n# Edit .env.local with your configuration\n\n# Set up the database\nnpm run db:generate\nnpm run db:push\n\n# Start the development server\nnpm run dev\n```\n\n### **Environment Variables**\n\n```env\n# Database\nDATABASE_URL=\"postgresql://...\"\n\n# Authentication\nNEXTAUTH_SECRET=\"your-secret-key\"\nNEXTAUTH_URL=\"http://localhost:3000\"\n\n# Supabase\nNEXT_PUBLIC_SUPABASE_URL=\"your-supabase-url\"\nNEXT_PUBLIC_SUPABASE_ANON_KEY=\"your-supabase-anon-key\"\n\n# Swiss Ephemeris (if using external service)\nSWISS_EPHEMERIS_API_KEY=\"your-api-key\"\n```\n\n## 📁 Project Structure\n\n```\nsrc/\n├── app/                    # Next.js App Router pages\n│   ├── astrology/         # Astrology
